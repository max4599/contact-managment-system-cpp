# contact-managment-system-cpp
Console application for managing contact records.
Features
	•	Add new contacts
	•	View stored contacts
	•	Edit existing records
	•	Delete contacts
	•	Save data to a file
	•	Load data from a file

Technologies
	•	C++
	•	File I/O
	•	Modular programming

Description

This project is a console-based contact management system developed in C++. The application allows users to create, view, edit, delete and store contact information using a menu-driven interface.
